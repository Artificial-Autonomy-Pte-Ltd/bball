CSV column format

Your sheet's header row must be exactly:

number, name, position, pos2, shooting, defense, playmaking, rebounding, iq, fitness, available, notes, hot_zones, type, parent, house

number = jersey # · position / pos2 = primary + optional secondary (PG/SG/SF/PF/C) · ratings 1–10 · available = TRUE/FALSE · hot_zones = comma list like "left-corner-3, top-of-key-3!" (! = love) · type = adult/child · parent = a child's parent jersey # · house = Morrison/Buckley/Hullett (blank = unassigned). Team kids use type=child + parent and skip ratings — they're excluded from lineups, rotation and the shooting board.
